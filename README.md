# ☕ Java Studies Repository

Este repositório reúne meus estudos em **Java**, com foco no desenvolvimento back-end, boas práticas de programação e fundamentos sólidos da linguagem.

## 📚 Objetivo

Consolidar conhecimentos em Java por meio de exercícios práticos, implementações e pequenos projetos, abordando desde conceitos básicos até estruturas mais utilizadas no mercado.

## 🧠 Conteúdos abordados

- Sintaxe básica do Java
- Programação Orientada a Objetos (POO)
  - Encapsulamento
  - Herança
  - Polimorfismo
  - Abstração
- Estruturas de controle
- Coleções (List, Set, Map)
- Manipulação de Strings
- Tratamento de exceções
- Entrada e saída de dados
- Boas práticas de código

## 🛠️ Tecnologias e ferramentas

- Java (JDK 8+)
- IDE (IntelliJ IDEA / Eclipse)
- Git & GitHub

## 📁 Estrutura do repositório

O repositório está organizado em arquivos e/ou pastas que representam diferentes tópicos ou exercícios:

javaEx/
├── src/
│ ├── exercicios/
│ ├── poo/
│ ├── collections/
│ └── ...
└── README.md


*(A estrutura pode evoluir conforme novos conteúdos são adicionados.)*

## 🚀 Como executar

1. Clone o repositório:
```bash
git clone https://github.com/Henrique8000/javaEx.git
```
1. Abra o projeto em sua IDE de preferência
2. Compile e execute os arquivos .java conforme necessário
